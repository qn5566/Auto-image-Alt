# Auto Image Alt Attribute WordPress Plugin

這是一個WordPress插件，它會自動為你的圖片設定alt屬性。

## 功能

- 自動為圖片設定alt屬性
- 使用者可以在WordPress後台自行設定alt屬性的值

## 安裝

1. 下載這個插件的.zip檔案
2. 在WordPress後台，前往"插件" > "新增" > "上傳插件"
3. 選擇你剛剛下載的.zip檔案，然後點擊"現在安裝"
4. 安裝完成後，點擊"啟用插件"

## 使用

1. 在WordPress後台，前往"設定" > "Auto Image Alt"
2. 在"Alt Attribute"欄位輸入你希望設定的alt屬性的值
3. 點擊"儲存變更"
